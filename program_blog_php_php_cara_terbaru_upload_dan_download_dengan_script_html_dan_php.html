PHP - Cara Terbaru Upload dan Download Dengan Script HTML dan PHP

Home 25 April 2018 Coding PHP

     Hay guys, Bertemu lagi nih. Terimakasih loh sudah mau mampir ke website saya. Sudah merelakan waktu anda untuk berkunjung kemari. Untuk anda yang hobinya dunia maya, ini nih pas banget tempat untuk menimba ilmu. Tempat yang cocok untuk kalian. Semua informasi saya menyediakannya di sini, dan itu kusus untuk anda.

     Kali ini saya akan membagikan sebuah informasi tentang Cara Terbaru Upload dan Download Dengan Script HTML dan PHP file yang akan masuk ke database dan di simpan di folder di dalam folder document root anda, dan file yang sudah di upload dapat di hapus. Berikut langkah-langkah pembuatannya :

  1. Pembuatan Database

    2. Langkah - langkahnya adalah sebagai berikut :

    1. Buat database dengan nama upload.
    2. Buat tabel dengan nama file.
    3. Selesai membuat database, lanjut ke langkah 2.

  2. Buat Folder

    3. Langkah - langkahnya adalah sebagai berikut :

    1. Buat Folder dengan nama updown dalam document root anda (contoh di htdocs pada MYSQL ).
    2. Simpan semua file yang nanti di buat di folder updown tersebut.
    3. Lalu buat folder dalam folder updown dan beri nama data (untuk menyimpan file yang nanti di upload).

  3. Membuat Script form upload

    4. Ketikan script berikut dan simpan dengan nama formupload.php :

    3. <html>
    4. <head>
    5. <title>Upload dan Download</title>
    6. </head>
    7. <body>
    8. <form enctype="multipart/form-data" action="ProsesUpload.php" method="POST">
    9. <input type="hidden" name="MAX_FILE_SIZE" value="3000000" />
    10. Pilih File: <input name="userfile" type="file" />
    11. <input type="submit" value="Upload" />
    12. <br>
    13. <a href="list.php">Lihat Daftar File</a>
    14. </form>
    15. <font color="red"><h5>causcript.com</h5></font>
    16. </body>
    17. </html>

  6. Buat script download

    7. ketikkan script berikut dan beri nama download.php

    3. <?php
    5. mysql_connect('localhost','root','');
    6. mysql_select_db('upload');
    8. $id = $_GET['id'];
    9. $query = "SELECT * FROM file WHERE id = '$id'";
    10. $hasil = mysql_query($query);
    11. $data = mysql_fetch_array($hasil);
    13. header("Content-Disposition: attachment; filename=".$data['name']);
    14. header("Content-length: ".$data['size']);
    15. header("Content-type: ".$data['type']);
    16. $fp = fopen("data/".$data['name'], 'r');
    17. $content = fread($fp, filesize('data/'.$data['name']));
    18. fclose($fp);
    19. echo $content;
    20. exit;
    21. ?>

  7. Buat Script hapus

    8. Ketikkan script berikut dan beri nama hapus.php

    3. <?php
    4. mysql_connect('localhost','root','');
    5. mysql_select_db('upload');
    7. $id = $_GET['id'];
    9. $query = "SELECT * FROM file WHERE id = '$id'";
    10. $hasil = mysql_query($query);
    11. $data = mysql_fetch_array($hasil);
    12. $namaFile = $data['name'];
    14. $query = "DELETE FROM file WHERE id = $id";
    15. mysql_query($query);
    17. unlink("data/".$namaFile);
    18. echo "File telah dihapus";
    19. ?>
    20. <br>
    21. <a href="list.php">Kembali Daftar File</a> <br>
    22. <a href="formupload.php">Kembali ke Formupload</a><br>
    23. <font color="red"><h5>causcript.com</h5></font>

  8. Uji coba Script

    1. Pergi ke http://localhost/updown/formupload.php.
    2. Klik Browse.., lalu pilih file yang akan di upload, setelah itu klik open.
    3. Setelah memilih file dan mengklik open, klik upload di formupload.
    4. Klik upload
    5. Cek di folder data dalam folder updown dan di database, apakah file sudah ada.
    6. Jika sudah ada maka proses upload file berhasil.
    7. Untuk mendownload atau menghapus file yang sudah di upload maka klik Daftar File di tampilan berhasil.
    8. Untuk mendownload klik download.
    9. Untuk menhapus klik Delete.
    10. Klik Kembali Daftar File, untuk ngecek data yang di hapus.
